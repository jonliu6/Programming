PowerOn Trouble Centre is a GIS system that displays outages in a list in one window and on a map in another window. It is the central repository for outage information. It allows dispatchers in the Trouble Centre to view and update the state of outages.

PowerOn Control Center is a GIS system that displays the state of each circuit in B.C. Hydro's network.

A web application that displays outages in a list. Dispatchers in remote offices collect information from service technicians and enter the information in PowerOn Remote. PowerOn Remote transfers the information to PowerOn Trouble Centre (POTC).

Service Link includes is two applications. One is a browser based system located in remote offices that organizes outages based on the service technician assigned to the outage. The other is a browser based system used in service technician vehicles to display the outages they are assigned to work on. The vehicle based system (mobile client) communicates with the office based system which then communicates with PowerOn Trouble Centre (POTC).

Plant Alteration Tracking is a web application to track PA and circuit changes from initiation in DAD to circuit rebuild in both PowerOn applications (POCC and POTC).

Emergency Services is a central source of information for the Emergency Services application.
