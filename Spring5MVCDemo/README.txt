This project shows some examples from "The Spring Web MVC Framework" at skillport.com - delivered by Janani Ravi

sample URLs for the view pages:
  http://localhost:8080/Spring5MVCDemo/sayHello
  http://localhost:8080/Spring5MVCDemo/exception?name=sql
  http://localhost:8080/Spring5MVCDemo/article/create
  http://localhost:8080/Spring5MVCDemo/uploadForm
  
The project is tested on Wildfly 15.0.1-Final, OpenJDL 11.0.2, Apache Maven 3.6.3
